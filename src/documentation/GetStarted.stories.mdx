### Overview

# Get Started

To get this package, make sure you turn on bhinneka <Code>VPN</Code> Install Tunggal complete with peer dependency.

To install the Tunggal through <Code>npm</Code>, run:

```bash
npm install tunggal --registry http://verdaccio.bhinnekalocal.com
```

To install the Tunggal through <Code>yarn</Code>, run:

```bash
yarn add tunggal --registry http://verdaccio.bhinnekalocal.com
```
